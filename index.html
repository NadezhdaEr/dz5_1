To-Do List

Создайте HTML-страницу, которая включает: Поле для ввода новой задачи (input). Кнопку "Добавить задачу". Список задач (ul). Кнопки для сортировки задач: "Показать все", "Показать выполненные", "Показать невыполненные". Функциональность: Добавление новой задачи. Удаление задачи. Отметка задачи как "выполненной" (задачи, которые выполнены, должны визуально отличаться). Возможность редактирования текста задачи. Сортировка задач по статусу (все задачи, выполненные, невыполненные). Фишки: Редактирование задачи по двойному клику на текст задачи. Визуальное выделение выполненных задач (например, зачёркивание текста).
